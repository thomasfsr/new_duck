## Passos para instalação:  
- Clonar o repositório:  
'git clone https://github.com/thomasfsr/new_duck/ '
- Iniciar o ambiente virtual:  
'poetry shell'  
- Instalar as dependencias com poetry:  
'poetry install'
- Iniciar o frontend com o taskipy:  
'task front'
- Documentação:
'task build_doc'
'task doc_run'
